# vimrc
Simple vimrc file designed to my liking

## Settings include
- UTF-8 encoding
- All tabs as spaces and set to four
- Autoindentation
- Ctrl-S saves the file (.bashrc must be configured)
- Auto closing of quotes and brackets
- Line numbers and textcolumn set at 80
